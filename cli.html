CLI

spapros

Create state of the art probesets for spatial transcriptomics projects.

spapros [OPTIONS] COMMAND [ARGS]...

Options

--version

Show the version and exit.

-v, --verbose

Enable verbose output (print debug statements).

-l, --log-file <log_file>

Save a verbose log to a file.

evaluation

Create a selection of probesets for an h5ad file.

Args:

data: Path to the h5ad dataset file probeset: Path to the probeset file marker_file: Path to the marker file probeset_ids: Several probeset ids parameters: Path to a yaml file containing parameters output: Output path

spapros evaluation [OPTIONS] DATA PROBESET MARKER_FILE [PROBESET_IDS]...

Options

--parameters <parameters>
-o, --output <output>

Arguments

DATA

Required argument

PROBESET

Required argument

MARKER_FILE

Required argument

PROBESET_IDS

Optional argument(s)

selection

Create a selection of probesets for an h5ad file.

Args:

data: Path to the h5ad file output: Output path

spapros selection [OPTIONS] DATA

Options

-o, --output <output>

Arguments

DATA

Required argument

upgrade

Checks whether the locally installed version of spapros is the latest & upgrades if not.

spapros upgrade [OPTIONS]

Example

Probeset selection

spapros selection data/small_data_raw_counts.h5ad

Probeset evaluation

spapros evaluation data/small_data_raw_counts.h5ad results/selections_genesets_1.csv data/small_data_marker_list.csv genesets_1_0 genesets_1_1 genesets_1_13 --parameters data/parameters.yml
