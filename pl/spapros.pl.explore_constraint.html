spapros.pl.explore_constraint

spapros.pl.explore_constraint(a, selections_tmp, penalty_kernels, x_axis_key='quantile_99', factors=None, upper=6, lower=2, size_factor=6, n_rows=1, legend_size=9, show=True, save=None)

Plot histogram of quantiles for selected genes for different penalty kernels.

Parameters

  • a (List[anndata._core.anndata.AnnData]) – List of sc.AnnData objects containing the data for plotting.

  • selections_tmp (List[pandas.core.frame.DataFrame]) – Dataframe containing the selection.

  • factors (Optional[List[float]]) – List of titles for the subplots, i.e. the factors of each penalty kernel.

  • upper (float) – Lower border above which the kernel is 1.

  • lower (float) – Upper boder below which the kernel is 1.

  • penalty_kernels (List[Callable]) – List of penalty kernels, which were used for selection.

  • x_axis_key (str) – Key of column in adata.var that is used for the x axis of the plotted histograms.

  • size_factor (int) – Factor for scaling the figure size.

  • n_rows (int) – Number of subplot rows.

  • legend_size (int) – Matplotlib legend size.

  • show (bool) – Whether to display the plot.

  • save (Optional[str]) – Save the plot to path.

Return type

None
